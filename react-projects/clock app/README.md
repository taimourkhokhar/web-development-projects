# 🕒 React Digital Clock

A clean, responsive, and real-time digital clock application built using React. This project demonstrates functional components, state management with React Hooks (`useState`, `useEffect`), and clean UI styling.


---

## ✨ Features

* **Real-Time Update:** Displays accurate time updating seamlessly every second.
* **12/24-Hour Format Toggle:** Switch between standard 12-hour (AM/PM) and 24-hour military time formats.
* **Smooth UI/UX:** Fully responsive design that scales beautifully on desktop, tablet, and mobile screens.
* **Zero Technical Debt:** Clean component architecture with no leftover boilerplate code.

---

## 🛠️ Tech Stack

* **Frontend:** React.js (Functional Components)
* **Hooks Used:** `useState` (for tracking time/format), `useEffect` (for managing the `setInterval` lifecycle)
* **Styling:** CSS3 / Flexbox
* **Build Tool:** Vite / Create React App
