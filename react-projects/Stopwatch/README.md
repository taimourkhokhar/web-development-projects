⏱️ React Stopwatch

A lightweight, accurate, and responsive stopwatch web application built with **React** (Hooks). It tracks elapsed time down to milliseconds using timestamps (`Date.now()`), avoiding the common time-drifting issues associated with standard intervals.

## ✨ Features

- **Start, Stop, & Reset:** Seamless state control for tracking time.
- **Millisecond Precision:** Displays time accurately in `MM:SS:MS` format.
- **Drift-Free Accuracy:** Tracks time differences dynamically rather than relying purely on the execution pacing of `setInterval`.
- **Clean Component Architecture:** Built using modern React Hooks (`useState`, `useEffect`, `useRef`).
